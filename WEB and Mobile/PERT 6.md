# PERT 6 - BOOTSTRAP

Bootstrap adalah framework CSS yang memudahkan kita untuk mengimplementasikan fitur-fitur website responsive, dan juga memudahkan kita untuk mendesain website modern dengan hanya menggunakan kelas-kelas HTML

Responsive dalam bootstrap adalah berdasarkan breakpoint, dimana breakpoint adalah nilai-nilai ukuran layar/window untuk menentukan layout objek. Bootstrap membagi layout dengan 12 ukuran sama besar

Ukuran Breakpoint Boostrap adalah berikut

|xs|sm|md|lg|xl|
|--|--|--|--|--|
|\< 576 px|\>= 576px|\>= 768 px|\>= 992 px|\>= 1200 px|

Beberapa tips Bootstrap adalah

1. Ada `-primary`,`-secondary`,`-warning`,`-danger`,`-success`, dsb yang dapat menentukan warna dari suatu komponen secara semantik
1. Ada beberapa kelas helper yang mungkin membantu dalam menambahkan fungsionalitas suatu situs web
1. Khusus untuk tag `<i>` ada kelas khusus untuk menambahkan ikon ikon pada Boostrap, jadi tidak selalu digunakan untuk teks *italic*
